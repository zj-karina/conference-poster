# ICML 2026 Workshop Poster — Authoritative Spec & Logistics

Source: icml.cc official poster instructions (2026). This skill targets the
**WORKSHOP** spec, which differs sharply from the main conference.

## Binding physical spec (WORKSHOP)

- **Orientation: PORTRAIT (mandatory).** Verbatim: "Workshop posters must be in
  portrait format."
- **Max size:** "Must not exceed 36in (H) × 24in (W) or 91cm (H) × 61cm (W)."
- Aspect ratio (w/h) = **0.667**. Many tools (e.g. PosterGen enforces w/h 1.4–2.0)
  cannot represent this without modification — this skill is portrait-locked.

## Do NOT use — main-conference sizes (landscape, wider)

The main conference suggests landscape sizes up to 36×72 in. These are wrong for a
workshop poster. Ignore main-track templates.

## Event & timing

- ICML 2026: **July 6–11, 2026**, COEX Convention & Exhibition Center, Seoul, Korea.
- July 6 = Expo/Tutorial; July 7–9 = main conference; **July 10–11 = workshops**.
- Workshop poster sessions are held in **Hall A**; organizers direct exact placement.

## Mounting (strict)

- Hang only on the **provided boards** in Hall A. Hanging on walls/public spaces at
  COEX is prohibited.
- Use **only conference-provided approved tape**. "Do not use adhesive backed
  self-sticking posters or anything that can damage poster boards." Bring nothing
  sticky of your own. (Main conference also provides push pins for its boards.)
- Confirm board location with the **workshop organizers** — they have final say.

## Printing

- Any service is fine. ICML partners with **The Foster / forceposter.com**.
- **Order deadline: June 15, 2026.** Pickup near Hall B2 (East Gate) at COEX.
- Recommended print resolution: **100–300 DPI at full size**.
- Export this poster as PDF at the true 24×36 in (≈1728×2592 pt) — see export_pdf.sh.

## Pre-print checklist

- [ ] Portrait, ≤ 24in W × 36in H.
- [ ] One giant plain-language takeaway, readable from across the room.
- [ ] QR code points to the correct arXiv/OpenReview/project URL (TEST IT).
- [ ] Every metric, equation, figure label, and author name proofread (AI can err).
- [ ] Figures ≥ ~150 DPI at print size — no blur.
- [ ] Ordered/printed before June 15; tape plan = conference tape only.
